# CONTEXT_PROJECT.md  
## Website – Официальное представительство МГУ (Украина) в Молдове

---

## 1. PROJECT OVERVIEW

This project is a **mobile-first educational website** for the **official representative / branch of MGU (Ukraine)** in Moldova.

The website’s main goal is to:
- Build **trust and legitimacy**
- Clearly communicate that we are an **official representative**
- Strongly present the **benefits of studying online**
- Convert users into **phone calls or WhatsApp / Telegram messages**

The website targets:
- **Students** (16–30 years old)
- **Parents** (decision makers)

Primary language: **Russian (RU)**  
Education language: **Russian (online)**

---

## 2. CORE IDEA & POSITIONING

We do NOT sell “just help with documents”.

We sell:
> **A safe, official, stress-free path to higher education at MGU, fully online, with personal support.**

Key differentiators:
- Official MGU representative
- 100% online education
- Official diploma (recognized in Moldova and other countries)
- Admission possible with low average grades
- Personal mentor for each student
- Full сопровождение (from consultation to enrollment)

Tone of voice:
- Semi-official
- Educational
- Friendly
- Trust-oriented
- Calm and reassuring (especially for parents)

---

## 3. MOBILE-FIRST APPROACH (CRITICAL)

This project MUST be designed **mobile-first**.

Workflow:
1. Design and finalize **mobile layout first**
2. Once a section is complete on mobile → adapt it for desktop
3. Desktop is an extension, not the base

UX principles:
- Short readable blocks
- Clear hierarchy
- Large tap targets
- Sticky contact actions on mobile
- Minimal cognitive load

---

This project is **UX-driven**, not visual-first.

Clarity > decorations  
Trust > animations  
Mobile usability > desktop complexity

> “Will a parent trust this?”  
> “Is this easy to understand on a phone?”
